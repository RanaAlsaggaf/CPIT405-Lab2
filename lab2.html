The objective of this course is to study Internet programming and Web application development

Students will learn basic principles and techniques for building Internet applications.

It provides students with the basic Web page development technologies and an introduction to dynamic Web page development1 using client-side scripting. Topics include introduction to HTTP protocol and client side programming.

XHTML, Cascading Style Sheets, and JavaScript DOM are core skills2 for building structured content. Students also learn how to manage layout and design using CSS3.

The course covers XML including Namespace, DTD and Schema, and introduces transformations with XSLT4. Students practice writing XML documents with attention to detaila.

Additional content includes XPATH and RSS feeds, where structured data is parsed and processed dynamically5. These technologies help students understand the flow of datab.

Finally, AJAX is introduced for asynchronous programming. With AJAX requests, Web applications can update parts of a page without refreshing6, providing a more responsive experiencec.
